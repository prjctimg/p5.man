# 📜 p5.man [![CI](https://github.com/prjctimg/p5.man/actions/workflows/update-docs.yml/badge.svg)](https://github.com/prjctimg/p5.man/actions/workflows/update-docs.yml)

> [!IMPORTANT]
> This repository is **not** intended for direct use. Use the generated documentation by installing the [p5.nvim](https://github.com/prjctimg/p5.nvim) plugin.

## How It Works

1. **Trigger**: GitHub Actions workflow runs when new releases are published to [processing/p5.js](https://github.com/processing/p5.js)
2. **Generate**: Clones p5.js source code and extracts documentation from JSDoc comments
3. **Format**: Converts documentation to neovim help file format with proper tags
4. **Deploy**: Updates [prjctimg/p5.nvim](https://github.com/prjctimg/p5.nvim) with generated files
